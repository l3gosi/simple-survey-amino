# simple-survey-amino
A simple script for survey-bot in Amino

## How to use?
1. Install Amino.Py library
2. Open `params.json` and edit fields:
* `"staff_chat"`: The id of the chat, where questions are going to be sent to (You will be able to get ids of all chats with `!listchats` command after running the bot, so you can skip this step)
* `"email"`: email of the bot account
* `"password"`: password of the bot account
* `"amino_name"`: amino id (for example: Anime)
3. Run `bot.py`
